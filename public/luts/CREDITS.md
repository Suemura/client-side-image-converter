# Preset LUT Credits / プリセット LUT のクレジット

The bundled preset LUTs in this directory (`*.cube`) come from two sources, both
usable under CC0. All files are renamed to clean, neutral names for the UI.

このディレクトリに同梱されているプリセット LUT（`*.cube`）は 2 つの出所からなり、
いずれも CC0 で利用可能なものです。UI 表示のため中立的な名称にリネームしています。

## 1. Algorithmically generated (original, CC0) / アルゴリズム生成（オリジナル・CC0）

Generated by `scripts/generate-luts.ts` in this repository. Completely original,
no third-party data. Released under **CC0 1.0 Universal**.

本リポジトリの `scripts/generate-luts.ts` が生成した完全なオリジナル（第三者データ非使用）。
**CC0 1.0 Universal** で提供します。

| File | UI Name | Description |
| --- | --- | --- |
| `cinematic.cube` | Cinematic / シネマ | Teal shadows / orange highlights with a mild contrast boost. |
| `warm.cube` | Warm / 暖色 | Warmer color temperature with a slight saturation lift. |
| `cool.cube` | Cool / 寒色 | Cooler color temperature. |
| `mono.cube` | Mono / モノクロ | Rec.709 luma grayscale with a gentle tone curve. |
| `vintage.cube` | Vintage / ヴィンテージ | Faded blacks with a subtle sepia cast and reduced saturation. |

Regenerate with `node scripts/generate-luts.ts`.
（`node scripts/generate-luts.ts` で再生成できます。）
