/ 色合い / 色相がモノクロ化前に効く B&W カラーフィルタ。彩度スライダーと独立）。値 0 の項目は GPU（uniform 分岐）/ CPU（フラグガード）とも計算を完全スキップ（既存プレビュー性能不変）。sharpness / grain は 0 起点の片方向スライダー（`ADJUSTMENT_UI_MIN`）。純粋ロジック
